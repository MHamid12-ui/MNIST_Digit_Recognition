# MNIST Digit Recognition AI
An end-to-end Machine Learning project that classifies handwritten digits (0-9) using a Convolutional Neural Network (CNN).

### Features
* **Model:** CNN built with TensorFlow/Keras.
* **Accuracy:** ~99% on the MNIST test set.
* **Interface:** Interactive Streamlit web app with a drawing canvas.

### How to use
1. Draw a digit in the black box.
2. Click 'Predict'.
3. The AI will show you its guess and confidence level.
